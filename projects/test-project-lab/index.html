Testing Demo Project Entry HTML Only

测试建议

  1. 回到首页，点击“添加项目卡片”。
  2. 项目名称输入：Test Project Lab。
  3. 项目简介输入：用于验证新增项目卡片、默认链接生成和跳转能力的测试页面。
  4. 项目重点输入：Validate add-project modal, local card rendering, and project page linking.
  5. 项目标签输入：Testing、Demo、HTML、Linking。
  6. 选择当前这个 test-project-lab 文件夹。
  7. 提交后点击“查看项目”，确认能跳到本页。
