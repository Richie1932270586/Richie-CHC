厂内物流助手

这是我基于物流实习期间整理的 SOP、FAQ 和异常规则做出的单 Agent demo。它保留了厂内物流场景最关键的能力组合：自然对话入口、RAG 检索、本地查表、动作草稿生成，以及高风险动作的人机确认。

单 Agent RAG Tools 人工确认 多会话聊天

接入方式

本页直接复用现有的 dist 聊天界面，默认会连接 http://127.0.0.1:8000。

  • 本地预览时，先启动 Agent 后端，再打开本页即可直接对话。
  • 如果后端不在本机 8000 端口，可在根目录 config.js 修改 FACTORY_AGENT_API_BASE。
  • 也可以通过 URL 参数 ?api=https://your-agent-host 临时覆盖连接地址。
cd "C:\Users\陈慧驰\Downloads\Richie-CHC (1)\Richie-CHC\projects\factory-exception-agent"
python -m uvicorn app.main:app --app-dir apps/api --host 127.0.0.1 --port 8000 --reload
